# Telegram Notifier

Telegram Notifier implementation in Node.js.

## Requirements

Node version >= 20

## Install dependencies

```sh
yarn install
```

## How to use

- Create the .env file based on the env-template.
- Starts the server by executing:
```sh
yarn serve:development
```
